《Perler-beads图片转拼豆图纸生成器PRD》

1. 产品名称
图片转拼豆图纸生成器

2. 产品定位
一款轻量、免费、开箱即用的在线拼豆辅助工具，用户上传任意图片，系统自动生成可直接对照制作的拼豆图纸，大幅降低拼豆爱好者的图纸设计门槛。

3. 目标用户
需要快速将照片转为拼豆图纸的用户

4. 核心参数
拼豆板规格
支持常见拼豆模板网格尺寸选择：
- 52cm×52cm
- 104cm×104cm
色板参数：
系统内置 MARD 官方标准色板，支持多档位切换：
- 24 色基础版
- 72 色标准版
- 168 色进阶版
- 221 色全色版

5. 拼豆图纸转化原理
通过HTML5 Canvas对上传图片进行处理，提取每个网格单元的主导色（出现频率最高的像素RGB值）
将其映射到最接近的MARD官方标准拼豆色（通过欧氏距离计算）
使用BFS算法将颜色相似的相邻单元格合并为区域，统一设置为区域内出现次数最多的色号，减少杂色并提升色块纯净度
根据用户选择的网格尺寸（52×52或104×104）和色板（24/72/168/221色），绘制带色号标记的网格
通过jsPDF库生成A4横向PDF图纸，确保输出结果清晰可打印

上传区

支持格式：JPG、PNG
